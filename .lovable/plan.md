

# Modelo de Mapeamento de Dados ERP - Template Visual Melhorado

## Objetivo
Transformar o documento atual de mapeamento (4 páginas genéricas em Word/PDF) em uma **aplicação web visual e estática** com fluxo claro, seções bem definidas e instruções detalhadas para a equipe de implantação preencher para cada cliente.

## Problemas identificados no modelo atual
- Tabelas genéricas sem exemplos concretos de preenchimento
- Sem indicação visual de ordem/fluxo entre as seções
- Campos vagos como "Detalhes" sem orientação do que escrever
- Seção de "Regras de Negócio" e "Telas do ERP" muito abertas, sem estrutura
- Não há checklist ou indicadores de progresso

---

## Estrutura da aplicação

### 1. **Página principal com navegação por etapas**
- Barra lateral ou stepper visual mostrando as 6 etapas do mapeamento em ordem
- Indicador visual de progresso (etapa atual destacada)
- Layout limpo com a marca Audaces

### 2. **Etapa 1 — Identificação do Projeto**
- Campos claros: Cliente, ERP Parceiro (com lista dos ERPs comuns da moda: TOTVS, Linx, Millennium, etc.), Implantador, Data de início
- Card com instruções do que preencher

### 3. **Etapa 2 — Cronograma de Implantação**
- Tabela pré-preenchida com etapas sugeridas (ex: Mapeamento, Desenvolvimento, Homologação, Go-live)
- Colunas: Etapa, Data Início, Data Conclusão, Status
- Dicas visuais em cada linha

### 4. **Etapa 3 — Benefícios Esperados**
- Área de texto com exemplos/placeholders das dores mais comuns (ex: "Retrabalho na digitação de fichas", "Falta de rastreabilidade de materiais")
- Checklist de benefícios comuns para marcar

### 5. **Etapa 4 — Fluxo da Ficha Técnica**
- Diagrama visual de fluxo com as etapas típicas: Criação → Desenvolvimento → Aprovação → Produção
- Tabela detalhada com colunas: Etapa, Sistema utilizado, Área responsável, Dados envolvidos, Observações
- Exemplo pré-preenchido como referência

### 6. **Etapa 5 — Telas do ERP Envolvidas**
- Tabela estruturada com colunas: Módulo do ERP, Tela/Cadastro, Campos preenchidos, Tipo de dado, Obrigatório (S/N)
- Categorias pré-definidas: Produto, Cores, Tamanhos, Materiais, Operações, Custos
- Espaço para screenshots/links

### 7. **Etapa 6 — Regras de Negócio e Documentos**
- Seção de regras com estrutura: Regra, Condição, Ação esperada, Sistema
- Seção de documentos com links organizados por tipo
- Checklist final de validação do mapeamento

### 8. **Design e UX**
- Visual profissional com cores da Audaces (azul/branco)
- Cards com ícones para cada seção
- Tooltips e textos de ajuda em cada campo
- Layout responsivo e otimizado para impressão/PDF
- Botão "Imprimir/Exportar" para gerar versão PDF
